# Issue tracker: GitHub

Issues and PRDs for this repo live in GitHub Issues for `OuyangWenyu/aqualord`.

## Conventions

- Create issues in GitHub Issues for this repository.
- Read issues, comments, labels, and state from GitHub Issues.
- Comment on existing work in the relevant GitHub issue before creating duplicate issues.
- Apply the triage labels defined in `docs/agents/triage-labels.md`.
- Infer the repository from `git remote -v` when running local GitHub tooling inside the clone.

## When a skill says "publish to the issue tracker"

Create a GitHub issue in `OuyangWenyu/aqualord`.

## When a skill says "fetch the relevant ticket"

Fetch the matching GitHub issue, including its body, comments, labels, and state.
